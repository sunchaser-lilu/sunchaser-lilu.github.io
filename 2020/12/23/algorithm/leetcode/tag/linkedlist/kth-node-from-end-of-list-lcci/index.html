题目：实现一种算法，找出单向链表中倒数第k个节点。返回该节点的值。

Leetcode在线OJ系统：传送门

示例：

graph LR
A[1]-->B[2]
B[2]-->C[3]
C[3]-->D[4]
D[4]-->E[5]

说明：给定的 k 保证是有效的。

思路：双指针法。定义两个指针first和second，first指针先走k - 1步，然后second指针和first指针同时走，每次走一步，当first指针到达链表末尾时，second指针指向的节点即为倒数第k个节点。
